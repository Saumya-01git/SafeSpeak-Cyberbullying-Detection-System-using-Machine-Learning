# 🛡️ SafeSpeak - Cyberbullying Detection System

SafeSpeak is a Machine Learning based web application that detects cyberbullying text using Natural Language Processing (NLP).

## 🚀 Features
- Detects cyberbullying messages
- Multiple ML models
- Accuracy comparison charts
- Interactive Streamlit UI
- Offensive keyword detection

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NLP

## 📊 Models Used
- Logistic Regression
- Naive Bayes
- SVM
- Random Forest

## ▶️ Run Locally

```bash
streamlit run app.py
